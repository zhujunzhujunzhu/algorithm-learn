# 栈

## 介绍
它是一种`后进先出`的数据结构。

## 基本方法
1. push 向栈中压入数据
2. pop  从栈中弹出数据
3. peek 拿到栈顶元素
4. clear 清空栈
5. size 获取栈的长度
6. isEmpty 判断栈是否为空
7. toString 栈的序列化

## 实现方式 
1. 基于数组的方式实现  O(n)
2. 基于javaScript的方式实现  O(1)

## 栈的使用场景
栈的应用非常广泛。在回溯问题中，它可以存储访问过的任务或路径，撤销的操作。java或c#用栈来存储变量和方法的调用，特别是在处理递归算法时，有可能抛出栈溢出异常。
在这里处理的应用是进行进制的转换。
